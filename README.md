## Descripción del proyecto

**BorrowMind** es un sistema de consola desarrollado en Python que permite gestionar el préstamo de diferentes tipos de objetos como herramientas, videojuegos y otros artículos.

El sistema contempla funcionalidades como:

- Registro y gestión de usuarios  
- Administración de inventario  
- Creación y seguimiento de préstamos  
- Control de fechas de devolución  
- Manejo de devoluciones  
- Notificaciones por incumplimiento  
- Generación de ventas automáticas en caso de mora  

---

## Objetivo

Desarrollar una aplicación de consola que permita administrar préstamos de manera estructurada, aplicando principios de **Programación Orientada a Objetos (POO)**, manejo de datos mediante archivos planos y buenas prácticas de desarrollo de software.
